CAPITOLO 01Che cos'è Atelier

Atelier è il tuo spazio di lavoro locale: apri, modifichi e organizzi i file di tutti i giorni senza mandare nulla su internet.

La maggior parte delle app oggi vive nel cloud: i tuoi documenti stanno sui server di qualcun altro, servono un account e una connessione. Atelier fa la scelta opposta. I tuoi file restano file, dentro una cartella del tuo disco che scegli tu. Niente account, niente sincronizzazione obbligata, niente abbonamenti: se un domani disinstalli Atelier, i tuoi documenti sono ancora lì, apribili con qualsiasi altro programma.

Il vault

Il vault è semplicemente la cartella che contiene il tuo lavoro. Può essere una cartella nuova o una che usi già, per esempio quella dei documenti o una cartella sincronizzata da OneDrive/Dropbox (la sincronizzazione, se la vuoi, la decidi tu con gli strumenti tuoi). Dentro il vault Atelier salva solo una piccola cartella di servizio .atelier con le impostazioni e il cestino.

Puoi avere più vault — ad esempio "Lavoro" e "Personale" — e passare dall'uno all'altro. Ogni finestra di Atelier apre un vault.

Un'app, tanti formati

Note — .md

Appunti in Markdown con la scrittura "ibrida" stile Obsidian, tabelle editabili e collegamenti tra note.

Documenti — .docx

Editor a pagine A4 vere, compatibile coi file di Word e Google Docs.

PDF

Lettura, ricerca, evidenziatore, moduli compilabili e OCR sulle scansioni.

Immagini — png, jpg, webp…

Visualizzazione e annotazioni: penna, frecce, forme, testo.

Fogli di calcolo — .xlsx, .csv

Griglia fedele a Excel con motore di formule, formattazione e 342 funzioni.

Presentazioni — .pptx

Visualizzatore fedele delle slide, miniature e modalità Presenta a schermo intero. L'editor è in arrivo.

CAPITOLO 02Primi passi

Aprire o creare un vault

Al primo avvio Atelier ti chiede dove vuoi lavorare:

  • Crea un vault — scegli dove creare una cartella nuova;
  • Apri una cartella — trasforma in vault una cartella esistente: i file che contiene compaiono subito nell'Explorer.

Dalla volta successiva trovi la lista dei vault conosciuti: un click e sei dentro. Se apri una seconda finestra di Atelier, la scelta del vault ricompare — così puoi tenere due vault aperti fianco a fianco.

La finestra

L'interfaccia è divisa in quattro zone, dall'alto in basso:

ZonaCosa contiene
Barra del titoloA sinistra il tasto che mostra/nasconde l'Explorer e il logo. Al centro puoi trascinare la finestra (doppio click = ingrandisci). A destra il selettore Standard / Developer e i controlli riduci · ingrandisci · chiudi. Se chiudi con modifiche non salvate, Atelier te lo chiede prima.
Riga delle tabUna scheda per ogni file aperto; a destra, il percorso della cartella del file attivo.
Explorer (sinistra)L'albero delle cartelle e dei file del vault.
Area di lavoroIl file aperto, con la sua barra degli strumenti.

L'Explorer

  • Le cartelle hanno la freccina (chevron) che ruota quando le apri.
  • I file mostrano il nome pulito e un badge con l'estensione (PDF, XLSX, JPG…); le note Markdown, il formato di casa, non hanno badge.
  • Trascina un file su una cartella per spostarlo.
  • La larghezza si regola trascinando il bordo destro; il tasto in alto a sinistra nella barra del titolo lo nasconde del tutto quando vuoi il massimo spazio per il contenuto.

Le tab

  • Ogni file aperto diventa una scheda: clicca per passare da un file all'altro.
  • Il pallino ambra sulla scheda segnala modifiche non ancora salvate.
  • Chiudi con la ✕ o col click centrale (la rotellina del mouse).
  • Riordina le schede trascinandole: una linea blu indica dove verrà inserita.
Suggerimento. I collegamenti tra note, i risultati della ricerca e i file convertiti si aprono tutti in una tab: non perdi mai quello su cui stavi lavorando.

CAPITOLO 03Lavorare coi file

Creare un file nuovo

Dal tasto Nuovo file dell'Explorer scegli il tipo (nota Markdown, documento Word, foglio di calcolo…), dai il nome e Atelier lo crea nella cartella selezionata e lo apre subito.

Portare dentro file esistenti

Il vault è una cartella normale: puoi copiarci dentro i file anche da Esplora risorse / Finder e compariranno nell'Explorer. Qualsiasi .docx, .xlsx, .pdf o immagine creato da altre app si apre così com'è.

Cercare

Due ricerche, sempre a portata di mano:

  • Ctrl+P — ricerca per nome: scrivi qualche lettera del nome del file, Invio e si apre.
  • Ctrl+Maiusc+F — ricerca globale nel contenuto: cerca una parola o una frase dentro i file di tutto il vault e salta dritto al risultato.

Rinominare, spostare, eliminare

Tutte le operazioni si fanno dall'Explorer. L'eliminazione non distrugge nulla: il file va nel Cestino del vault, da cui puoi recuperarlo. Le tab aperte seguono i file: rinomini o sposti un file e la sua scheda si aggiorna da sola.

Convertire da un formato all'altro

Quando un file si può trasformare, nella sua barra degli strumenti compare il tasto Converti. La conversione crea un file nuovo accanto all'originale (l'originale non viene mai toccato) e lo apre subito.

CAPITOLO 04Note e appunti (Markdown)

Il formato di casa di Atelier: file di puro testo che restano tuoi per sempre, con una scrittura che diventa bella da leggere mentre la scrivi.

Le tre viste

In alto scegli come vedere la nota:

VistaQuando usarla
EditorIl sorgente Markdown con l'evidenziazione della sintassi. Per chi vuole il controllo totale sul testo.
IbridaLa via di mezzo perfetta, stile Obsidian: scrivi Markdown e la riga si trasforma nel risultato appena la lasci. Titoli grandi, grassetti veri, tabelle vere.
LetturaSolo il risultato finale, impaginato con i titoli in carattere serif. Per leggere e presentare.

Cosa puoi scrivere

  • Titoli (# Titolo), grassetto, corsivo, elenchi puntati e numerati, citazioni, righe di separazione.
  • Tabelle editabili: nella vista Ibrida le tabelle Markdown diventano vere tabelle — clicchi nella cella e scrivi, e dal menu aggiungi o togli righe e colonne. Niente più trattini da allineare a mano.
  • Collegamenti tra note con [[nome della nota]]: click e ti sposti; la nota si apre nella sua tab.
  • Callout — riquadri colorati per note e avvisi (> [!note]).
  • Blocchi di codice con l'evidenziazione per decine di linguaggi.
  • Immagini incorporate nella nota.

Menu contestuale

Il tasto destro nell'editor offre le azioni rapide sul testo selezionato: formattazione, collegamenti e operazioni sulle tabelle.

Perché Markdown? È testo semplice: si legge con qualsiasi programma, si versiona, si cerca, non si corrompe. Le tue note di oggi si apriranno identiche tra vent'anni.

CAPITOLO 05Documenti Word

Un editor a pagine vere: quello che vedi sullo schermo è il foglio A4 che stamperai, senza aprire Word.

Compatibilità

Atelier apre i .docx creati da Microsoft Word, Google Docs, LibreOffice e li salva restando compatibile: il file rimane un normale .docx apribile ovunque.

Scrivere e formattare

  • Carattere: grassetto, corsivo, sottolineato, colore del testo, evidenziazione, famiglia del font, apice e pedice.
  • Paragrafo: titoli, allineamento (sinistra/centro/destra/giustificato), elenchi puntati e numerati, liste di attività con casella di spunta.
  • Tabelle con inserimento e gestione di righe e colonne.
  • Immagini dentro il documento, dal tasto dedicato.

Impostazioni del documento

Dal pannello Impostazioni regoli il formato della pagina e i margini. Con la vista continua le pagine scorrono senza interruzioni, comoda per la scrittura lunga; disattivala per vedere i salti pagina reali.

Salvare

Il tasto Salva (in evidenza, col gradiente blu) scrive il .docx. Il pallino ambra sulla tab ti ricorda quando ci sono modifiche in sospeso.

CAPITOLO 06PDF

Leggere, trovare, evidenziare e compilare — con una rete di sicurezza automatica prima di ogni modifica.

Lettura e ricerca

Zoom, scorrimento fluido pagina per pagina e ricerca nel testo del documento dal tasto Cerca. Il pannello Info mostra i metadati (titolo, autore, numero di pagine…).

Evidenziatore

Attiva l'evidenziatore, scegli il colore dalla palette e passa sul testo: le evidenziazioni vengono salvate dentro il PDF, quindi le ritrovi anche aprendo il file con qualunque altro lettore.

Alla prima modifica Atelier crea automaticamente una copia di backup (.bak) accanto al file: l'originale è sempre recuperabile.

Moduli compilabili

Se il PDF contiene un modulo (campi di testo, caselle…), i campi sono evidenziati e si compilano direttamente; il salvataggio scrive i valori nel file, pronti da inviare.

OCR — il testo delle scansioni

Un PDF scannerizzato è solo un'immagine: con l'OCR Atelier ne estrae il testo, che puoi copiare e riusare. Tutto avviene sul tuo computer — nessun documento viene caricato da nessuna parte.

CAPITOLO 07Immagini

Guardare da vicino e spiegare con un segno: frecce, evidenziazioni e note direttamente sull'immagine.

Visualizzazione

Zoom con la rotellina e pan con lo strumento mano per muoverti nell'immagine ingrandita. Il pannello Info mostra dimensioni e dettagli del file.

Annotazioni

La barra degli strumenti offre tutto il necessario per marcare un'immagine:

StrumentoCosa fa
SelezioneRiprendi un elemento già disegnato: lo puoi spostare, ridimensionare o cancellare con le maniglie blu.
Penna 1 e 2Disegno a mano libera in due spessori, colore a scelta.
FrecciaFrecce dritte per indicare un punto.
FormeRettangoli ed ellissi, per incorniciare una zona.
TestoEtichette di testo sull'immagine.
GommaRimuove i singoli elementi disegnati.

Finché non premi Applica, le annotazioni restano modificabili: puoi sistemarle, spostarle, cambiarle. Applica le fonde nell'immagine e salva.

Qualità preservata

Il salvataggio mantiene i DPI originali del file: un'immagine da stampa resta da stampa. E con l'OCR estrai il testo fotografato (cartelli, documenti, lavagne).

CAPITOLO 08Fogli di calcolo

Se sai usare Excel, sai già usare Atelier: stessa logica, stessi tasti, stesse formule. Cambiano solo i colori.

I tuoi file, intatti

Atelier apre .xlsx e .csv e li preserva per intero: tutto ciò che il file contiene — anche le cose che Atelier non mostra ancora, come i grafici — sopravvive al salvataggio. Il foglio che riapri in Excel è lo stesso che hai salvato.

La griglia

  • Larghezze e altezze reali, celle unite, colori, bordi e formati numero come nel file originale.
  • Fogli multipli con le linguette in basso: aggiungi, rinomina, elimina (con conferma).
  • Migliaia di righe senza rallentamenti: la griglia disegna solo ciò che vedi.
  • Blocca riquadri per tenere ferme intestazioni e prime colonne (dal menu del tasto destro; l'impostazione viene scritta nel file).
  • Filtri sulle intestazioni e righe nascoste rispettate.
  • La barra di stato in basso mostra Somma, Media e Conteggio della selezione — come in Excel.

Muoversi e selezionare

  • Click su una cella = selezione; trascina per selezionare un intervallo; Shift+click per estendere.
  • Click sui numeri di riga o le lettere di colonna = selezione dell'intera riga/colonna; trascina sulle intestazioni per selezionarne più d'una.
  • Ctrl+frecce salta al bordo dei dati, Ctrl+F trova nel foglio.

Scrivere nelle celle

  • Seleziona e scrivi: il contenuto viene sostituito (come Excel). Per modificare il contenuto esistente usa F2 o il doppio click.
  • Invio conferma e scende, Tab conferma e va a destra, Esc annulla.
  • Date (12/5), orari (9:30) e percentuali (25%) digitati vengono riconosciuti e ricevono il formato giusto da soli.
  • Canc apre il mini-menu: solo contenuto, solo formattazione o tutto. Backspace svuota il contenuto senza domande.
  • Annulla e ripeti (Ctrl+Z / Ctrl+Y) valgono su tutto: valori, formati e anche le operazioni su righe e colonne.

Copia, incolla e riempimento

  • Ctrl+C/X/V con stili e formule: le formule incollate traslano i riferimenti (come Excel); il Taglia svuota l'origine al momento dell'incolla.
  • Il quadratino di riempimento nell'angolo della selezione trascina serie (1, 2, 3… / lun, mar…) e formule; il doppio click sul quadratino riempie fino in fondo alla colonna vicina.
  • Ctrl+D riempie in giù, Ctrl+R a destra.
  • Trascina il bordo della selezione per spostare le celle.

Formattare

  • Dalla barra: grassetto, corsivo, sottolineato, barrato, dimensione del carattere, colore del testo e dello sfondo, allineamento.
  • Formato celle: bordi lato per lato con stile e colore, e i formati numero (valuta, percentuale, date, ore, decimali…).
  • Stili tabella pronti: intestazione piena e righe alternate in un click.
  • Le modifiche di formato valgono per la sola selezione — e naturalmente si annullano con Ctrl+Z.

Righe, colonne e dati

  • Tasto destro sulle intestazioni: inserisci o elimina righe e colonne, anche più d'una alla volta ("Elimina 3 righe"). Le formule del foglio si traslano da sole, con la stessa semantica di Excel.
  • Doppio click sul bordo di una colonna = larghezza automatica sul contenuto.
  • Ordinamento crescente/decrescente, numerico e alfabetico.
  • Unione celle dalla barra degli strumenti.

CSV

I file .csv si aprono nella stessa griglia, si modificano e si salvano; dal tasto Converti li trasformi in .xlsx veri quando ti servono formule e formati.

CAPITOLO 09Le formule

Un motore di calcolo vero: 342 funzioni, nomi in italiano o in inglese, ricalcolo immediato a catena.

Come si scrive una formula

Come in Excel: selezioni la cella, scrivi = e componi. Puoi usare i nomi italiani o inglesi — =SOMMA(A1:A5) e =SUM(A1:A5) sono la stessa cosa (nel file viene sempre salvato il nome inglese, esattamente come fa Excel, così il foglio si apre ovunque). Funzionano il ; come separatore degli argomenti e la virgola decimale.

  • La barra della formula in alto mostra l'indirizzo della cella e il suo contenuto vero.
  • Mentre scrivi, l'autocompletamento suggerisce le funzioni (=SO → SOMMA, SOSTITUISCI…).
  • In modalità formula, clicca una cella (o trascina un intervallo) per inserirne il riferimento: ogni riferimento è colorato, col riquadro dello stesso colore sulla cella — leggi la formula a colpo d'occhio.
  • F4 cicla i riferimenti assoluti: A1 → $A$1 → A$1 → $A1.
  • Cambi un valore e tutte le formule che ne dipendono si aggiornano, anche in cascata e tra fogli diversi. I risultati vengono scritti nel file: chi lo apre con un'altra app vede i numeri giusti.

Limiti attuali (detti onestamente)

  • SCARTO/OFFSET e INDIRECT non vengono ricalcolate in diretta (richiedono riferimenti dinamici): la cella mostra il valore salvato nel file.
  • Le funzioni "dinamiche" di Excel 365 (FILTER, SORT, SEQUENCE, LAMBDA…) non ci sono ancora; UNIQUE c'è in forma base.
  • Se un foglio usa una funzione non supportata non si rompe nulla: la cella conserva l'ultimo valore calcolato da Excel, e il file resta integro.

CAPITOLO 11Domande frequenti

Dove sono i miei file?

Nella cartella del vault, come file normali. Puoi aprirli, copiarli e farne il backup con gli strumenti che già usi. Atelier non li converte e non li nasconde in nessun database.

Serve una connessione a internet?

No. Tutto — modifica, formule, OCR, conversioni — avviene sul tuo computer.

Se elimino un file per sbaglio?

È nel Cestino del vault (cartella .atelier): recuperalo da lì.

Posso aprire i file di Atelier con altre app?

Sì, sempre: .docx, .xlsx, .pdf e le immagini restano file standard al 100%. Le note sono testo Markdown, leggibile ovunque.

Excel riaprirà bene i fogli che modifico?

Sì: Atelier preserva l'intero contenuto del file, comprese le parti che non mostra (ad es. i grafici). Le formule sono salvate coi nomi standard e i risultati aggiornati.

E le presentazioni PowerPoint?

I .pptx si aprono con un visualizzatore fedele (forme, testi, immagini, SmartArt): miniature nella barra laterale, zoom e il tasto Presenta per lo schermo intero — click o freccia destra per avanzare, freccia sinistra per tornare indietro, Esc per uscire. La modifica delle slide è in lavorazione.

Cosa manca ancora?

L'editor delle presentazioni (oggi si aprono in sola lettura), i grafici di Excel non vengono ancora disegnati, e alcune funzioni dinamiche di Excel 365 non sono supportate (vedi il capitolo 9).
