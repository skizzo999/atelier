CHAPTER 01What Atelier is

Atelier is your local workspace: open, edit and organise your everyday files without sending anything to the internet.

Most apps today live in the cloud: your documents sit on someone else's servers, and you need an account and a connection. Atelier makes the opposite choice. Your files stay files, inside a folder on your own disk that you choose. No account, no forced sync, no subscriptions: if you ever uninstall Atelier, your documents are still there, ready to open with any other program.

The vault

The vault is simply the folder that holds your work. It can be a brand-new folder or one you already use — your documents folder, or a folder synced by OneDrive/Dropbox (syncing, if you want it, stays your choice with your own tools). Inside the vault Atelier only keeps a small service folder, .atelier, with settings and the trash.

You can have several vaults — say "Work" and "Personal" — and switch between them. Each Atelier window opens one vault.

One app, many formats

Notes — .md

Markdown notes with Obsidian-style hybrid writing, editable tables and links between notes.

Documents — .docx

An editor with real A4 pages, compatible with Word and Google Docs files.

PDF

Reading, search, highlighter, fillable forms and OCR for scans.

Images — png, jpg, webp…

Viewing and annotations: pen, arrows, shapes, text.

Spreadsheets — .xlsx, .csv

An Excel-faithful grid with a formula engine and 342 functions.

Presentations — .pptx

Faithful slide viewer with thumbnails and a full-screen Present mode. Editing is on the way.

CHAPTER 02First steps

Opening or creating a vault

On first launch Atelier asks where you want to work:

  • Create a vault — choose where to create a new folder;
  • Open a folder — turn an existing folder into a vault: the files it contains appear straight away in the Explorer.

From the next launch you get the list of known vaults: one click and you're in. If you open a second Atelier window, the vault picker appears again — so you can keep two vaults open side by side.

The window

The interface is divided into four areas, top to bottom:

AreaWhat it holds
Title barOn the left, the button that shows/hides the Explorer and the logo. In the middle you can drag the window (double click = maximise). On the right, the Standard / Developer switch and the minimise · maximise · close controls. If you close with unsaved changes, Atelier asks first.
Tab rowOne tab per open file; on the right, the folder path of the active file.

The Explorer

  • Folders have a chevron that rotates when you open them.
  • Files show a clean name and a badge with the extension (PDF, XLSX, JPG…); Markdown notes, the home format, carry no badge.
  • Drag a file onto a folder to move it.
  • Resize it by dragging its right edge; the button at the top left of the title bar hides it entirely when you want maximum room for content.

Tabs

  • Every open file becomes a tab: click to switch between files.
  • The amber dot on a tab flags unsaved changes.
  • Close with the ✕ or a middle click (the mouse wheel).
  • Reorder tabs by dragging them: a blue line shows where the tab will land.
Tip. Links between notes, search results and converted files all open in a tab: you never lose what you were working on.

CHAPTER 03Working with files

Creating a new file

From the Explorer's New file button pick the type (Markdown note, Word document, spreadsheet…), give it a name, and Atelier creates it in the selected folder and opens it immediately.

Bringing in existing files

The vault is a normal folder: you can copy files into it from File Explorer / Finder and they show up in the Explorer. Any .docx, .xlsx, .pdf or image created by other apps opens as-is.

Searching

Two searches, always at hand:

  • Ctrl+P — search by name: type a few letters of the file name, hit Enter and it opens.
  • Ctrl+Shift+F — global content search: find a word or phrase inside the files of the whole vault and jump straight to the match.

Renaming, moving, deleting

All of it happens in the Explorer. Deleting never destroys anything: the file goes to the vault's Trash, where you can recover it. Open tabs follow their files: rename or move a file and its tab updates by itself.

Converting between formats

Whenever a file can be transformed, its toolbar shows a Convert button. Conversion creates a new file next to the original (the original is never touched) and opens it right away.

CHAPTER 04Notes (Markdown)

Atelier's home format: plain-text files that stay yours forever, with writing that turns beautiful to read as you type it.

The three views

At the top you choose how to see the note:

ViewWhen to use it
EditorThe Markdown source with syntax highlighting. For full control over the text.
HybridThe perfect middle ground, Obsidian-style: you write Markdown and the line turns into the result the moment you leave it. Big headings, real bold, real tables.
ReadingOnly the final result, typeset with serif headings. For reading and presenting.

What you can write

  • Headings (# Title), bold, italic, bulleted and numbered lists, quotes, dividers.
  • Editable tables: in the Hybrid view Markdown tables become real tables — click a cell and type, and use the menu to add or remove rows and columns. No more aligning dashes by hand.
  • Links between notes with [[note name]]: click to jump; the note opens in its own tab.
  • Callouts — coloured boxes for notes and warnings (> [!note]).
  • Code blocks with highlighting for dozens of languages.
  • Images embedded in the note.

Context menu

Right-click in the editor for quick actions on the selected text: formatting, links and table operations.

Why Markdown? It's plain text: readable with any program, versionable, searchable, incorruptible. Today's notes will open identical twenty years from now.

CHAPTER 05Word documents

An editor with real pages: what you see on screen is the A4 sheet you'll print, without opening Word.

Compatibility

Atelier opens .docx files created by Microsoft Word, Google Docs and LibreOffice, and saves them still compatible: the file remains a normal .docx you can open anywhere.

Writing and formatting

  • Character: bold, italic, underline, text colour, highlight, font family, superscript and subscript.
  • Paragraph: headings, alignment (left/centre/right/justified), bulleted and numbered lists, task lists with checkboxes.
  • Tables with row and column management.
  • Images inside the document, from the dedicated button.

Document settings

The Settings panel controls page format and margins. With the continuous view pages scroll without breaks — great for long-form writing; switch it off to see the real page breaks.

Saving

The Save button (highlighted with the blue gradient) writes the .docx. The amber dot on the tab reminds you when changes are pending.

CHAPTER 06PDF

Read, find, highlight and fill in — with an automatic safety net before any change.

Reading and search

Zoom, smooth page-by-page scrolling and text search from the Search button. The Info panel shows the metadata (title, author, page count…).

Highlighter

Turn on the highlighter, pick a colour from the palette and sweep over the text: highlights are saved inside the PDF, so you'll find them again in any other reader.

On the first change Atelier automatically creates a backup copy (.bak) next to the file: the original is always recoverable.

Fillable forms

If the PDF contains a form (text fields, checkboxes…), the fields are highlighted and can be filled in directly; saving writes the values into the file, ready to send.

OCR — text from scans

A scanned PDF is just a picture: with OCR Atelier extracts its text so you can copy and reuse it. Everything happens on your computer — no document is uploaded anywhere.

CHAPTER 07Images

Look closely and explain with a mark: arrows, highlights and notes drawn straight onto the image.

Viewing

Zoom with the wheel and pan with the hand tool to move around a zoomed-in image. The Info panel shows the file's size and details.

Annotations

The toolbar has everything you need to mark up an image:

ToolWhat it does
SelectPick up an element you've already drawn: you can move, resize or delete it with the blue handles.
Pen 1 and 2Freehand drawing in two thicknesses, any colour.
ArrowStraight arrows to point at things.
ShapesRectangles and ellipses, to frame an area.
TextText labels on the image.
EraserRemoves individual drawn elements.

Until you press Apply, annotations stay editable: adjust them, move them, change them. Apply merges them into the image and saves.

Quality preserved

Saving keeps the file's original DPI: a print-quality image stays print-quality. And with OCR you can extract photographed text (signs, documents, whiteboards).

CHAPTER 08Spreadsheets

If you know Excel, you already know Atelier: same logic, same keys, same formulas. Only the colours are different.

Your files, intact

Atelier opens .xlsx and .csv files and preserves them entirely: everything the file contains — even things Atelier doesn't display yet, like charts — survives saving. The sheet you reopen in Excel is the one you saved.

The grid

  • Real column widths and row heights, merged cells, colours, borders and number formats as in the original file.
  • Multiple sheets with tabs at the bottom: add, rename, delete (with confirmation).
  • Thousands of rows without slowdown: the grid draws only what you see.
  • Freeze panes to pin headers and first columns (from the right-click menu; the setting is written into the file).
  • Filters on headers, hidden rows respected.
  • The status bar shows Sum, Average and Count for the selection — just like Excel.

Moving and selecting

  • Click a cell = selection; drag to select a range; Shift+click to extend.
  • Click the row numbers or column letters to select whole rows/columns; drag across the headers to select several.
  • Ctrl+arrows jumps to the edge of the data, Ctrl+F finds in the sheet.

Typing into cells

  • Select and type: the content is replaced (like Excel). To edit the existing content use F2 or double click.
  • Enter confirms and moves down, Tab confirms and moves right, Esc cancels.
  • Typed dates (12/5), times (9:30) and percentages (25%) are recognised and get the right format automatically.
  • Del opens the mini-menu: content only, formatting only or everything. Backspace clears the content, no questions asked.
  • Undo and redo (Ctrl+Z / Ctrl+Y) cover everything: values, formats and even row/column operations.

Copy, paste and fill

  • Ctrl+C/X/V with styles and formulas: pasted formulas shift their references (like Excel); Cut clears the source at paste time.
  • The fill handle in the selection's corner drags series (1, 2, 3… / Mon, Tue…) and formulas; double click it to fill down as far as the neighbouring column.
  • Ctrl+D fills down, Ctrl+R fills right.
  • Drag the selection's border to move cells.

Formatting

  • From the toolbar: bold, italic, underline, strikethrough, font size, text and fill colour, alignment.
  • Format cells: per-side borders with style and colour, and number formats (currency, percentage, dates, times, decimals…).
  • Ready-made table styles: solid header and banded rows in one click.
  • Formatting applies to the selection only — and of course undoes with Ctrl+Z.

Rows, columns and data

  • Right-click the headers to insert or delete rows and columns, several at a time too ("Delete 3 rows"). The sheet's formulas shift by themselves, with Excel's exact semantics.
  • Double click a column edge = auto-fit width to content.
  • Sorting ascending/descending, numeric and alphabetical.
  • Merge cells from the toolbar.

CSV

.csv files open in the same grid, can be edited and saved; the Convert button turns them into real .xlsx when you need formulas and formatting.

CHAPTER 09Formulas

A real calculation engine: 342 functions, instant chained recalculation, English and Italian function names.

Writing a formula

Just like Excel: select the cell, type = and build. Function names are the standard English ones (=SUM(A1:A5)); Italian names are accepted too (=SOMMA(A1:A5) is the same formula — the file always stores the English name, exactly as Excel does, so the sheet opens anywhere). Both , and ; work as argument separators.

  • The formula bar at the top shows the cell's address and its true content.
  • As you type, autocomplete suggests functions (=SU → SUM, SUBSTITUTE…).
  • In formula mode, click a cell (or drag a range) to insert its reference: every reference is colour-coded, with a matching outline on the cell — you read the formula at a glance.
  • F4 cycles absolute references: A1 → $A$1 → A$1 → $A1.
  • Change a value and every dependent formula updates, in cascade and across sheets. Results are written into the file: anyone opening it with another app sees the right numbers.

Current limits (stated honestly)

  • OFFSET and INDIRECT are not recalculated live (they require dynamic references): the cell shows the value stored in the file.
  • Excel 365's "dynamic array" functions (FILTER, SORT, SEQUENCE, LAMBDA…) are not available yet; UNIQUE exists in a basic form.
  • If a sheet uses an unsupported function nothing breaks: the cell keeps the last value Excel calculated, and the file stays intact.

CHAPTER 11Frequently asked questions

Where are my files?

In the vault folder, as normal files. You can open them, copy them and back them up with the tools you already use. Atelier doesn't convert them and doesn't hide them in any database.

Do I need an internet connection?

No. Everything — editing, formulas, OCR, conversions — happens on your computer.

What if I delete a file by mistake?

It's in the vault's Trash (the .atelier folder): recover it from there.

Can I open Atelier's files with other apps?

Always: .docx, .xlsx, .pdf and images remain 100% standard files. Notes are Markdown text, readable everywhere.

Will Excel reopen the sheets I edit correctly?

Yes: Atelier preserves the file's entire content, including the parts it doesn't display (e.g. charts). Formulas are stored with their standard names and up-to-date results.

What about PowerPoint presentations?

.pptx files open in a faithful viewer (shapes, text, images, SmartArt) with thumbnails in the sidebar, zoom, and a Present button for full screen — click or right arrow to advance, left arrow to go back, Esc to leave. Slide editing is in the works.

What's still missing?

The presentation editor (decks open read-only for now), Excel charts aren't drawn yet, and some Excel 365 dynamic functions aren't supported (see chapter 9).
