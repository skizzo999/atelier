# Sessione 2026-06-22 (8) - Crop immagini + CodeMirror 6

## Obiettivo
Due feature in sequenza, in autonomia: ritaglio immagini (fase 1b) e
integrazione di CodeMirror 6 come editor sorgente.

## Crop immagini (fase 1b)
- Modalità ritaglio in EditableImage: si trascina un rettangolo sull'immagine,
  "Applica ritaglio" ritaglia il canvas alla regione selezionata.
- Overlay che cattura il mouse (absolute inset-0); rettangolo di selezione con
  dim esterno via box-shadow; durante il crop si forza il "fit" per coordinate stabili.
- Mappatura coordinate display→pixel con getBoundingClientRect (scaleX/scaleY).
- Riusa applyTransform → canvas + buffer immagini + dirty + salvataggio atomico.

## CodeMirror 6 (editor sorgente)
- src/components/CodeMirror/CodeMirrorEditor.tsx: bridge React↔CM6.
  - value = fonte di verità React; modifiche utente → onChange; cambi esterni di
    value (cambio file) applicati al doc senza ri-emettere onChange (flag guard).
  - Estensioni: history, defaultKeymap+historyKeymap+indentWithTab, lineWrapping,
    oneDark, markdown() (solo per file markdown), tema base (altezza/font).
  - Ricreato solo se cambia la modalità markdown/plain; altrimenti aggiorna il doc.
- Editor: la vista "Codice" usa CodeMirrorEditor al posto della textarea.
  - Highlight della ricerca adattato a CM6 (dispatch selection + scrollIntoView).
  - Ctrl+S, dirty/buffer, vista "Lettura" (marked+DOMPurify) invariati.
- Dipendenze: @codemirror/{state,view,commands,language,lang-markdown,theme-one-dark}.

## Note / limiti
- CM6 è la **base** dell'editor: la vista "Ibrida"/live-preview (decorazioni inline
  su CM6) è il layer successivo, non ancora fatta.
- Tema CM6 oneDark: sfondo leggermente diverso dall'app (zinc). Si uniforma nella
  fase grafica (token colore).
- Bundle > 500kB (warning): eventuale code-split di CM6 più avanti.

## Aggiunto dopo: vista Ibrida (live preview)
- src/components/CodeMirror/livePreview.ts: ViewPlugin che costruisce decorazioni
  dall'albero sintattico: nasconde i marcatori (Decoration.replace) e formatta il
  contenuto (Decoration.mark) per titoli (ATXHeading1-6), grassetto (StrongEmphasis),
  corsivo (Emphasis), inline code, link. La riga con il cursore mostra il grezzo
  (active line). try/catch → degrada a nessuna decorazione in caso di errore.
- CodeMirrorEditor: prop livePreviewMode (ricrea la view quando cambia).
- Editor: terzo toggle "Ibrida" (view: source | live | reading); highlight ricerca
  attivo anche in live.
- Crop: aggiunta scelta "Crea nuova foto" (nuovo file -ritaglio) vs "Applica".
